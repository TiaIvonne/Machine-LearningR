Un mtodo de seleccin de variables es utilizando el algoritmo de random forest para encontrar un set de predictores, para eso se utiliza la librera party y se obtienen los siguientes resultados:

library(party)

Utilizando MARS: Multivariate Adaptive Regression Splines

Otro mtodo de seleccin de variables es utilizando MARS, contenida en la librera earth para R

library(earth)
s2 <- earth(varObjBin ~ ., data=dengue2) # build model
evaluacion <- evimp (s2)
plot(evaluacion)

Utilizando la libreria Boruta

Boruta es otra librera para realizar feature selection de forma automtica, permitiendo un acercamiento inicial rpido al dataset (con sus ventajas y desventajas):

library(Boruta)
boruta_model <- Boruta(varObjBin ~ ., data= dengue2, doTrace=2)  
relevancia <- names(boruta_model$finalDecision[boruta_model$finalDecision %in% c("Confirmed")])  
print(relevancia) 
##  [1] "humid"    "humid90"  "temp"     "temp90"   "h10pix"   "h10pix90"
##  [7] "trees"    "trees90"  "Xmin"     "Xmax"     "Ymin"     "Ymax"
plot(boruta_model, las = 2, cex.axis=0.7, xlab="")

Boruta permite revisar si el set de variables a estudiar entran en el modelo o podran entrar de forma tentativa o definitivamente no han sido considerados como variables predictoras relevantes con lo cual tambin se pueden probar otros modelos y combinaciones (ej confirmados mas algn tentativo etc)

Para este dataset en particular la seleccin no marca variables como tentativas o descartadas por lo cual correspondera observar el grfico generado y realizar algn corte de tipo manual en las variables.

Utilizando RFE con caret

Eliminacin recursiva de caractersticas o RFE en sus siglas en ingles, es un estimador que asigna pesos a las caractersticas del dataset que se quiere estudiar

set.seed(7)
# Cargar datos
data(dengue2)
# Se define la funcion de control
controlrfe <- rfeControl(functions=rfFuncs, method="cv", number=4)
# Algoritmo RFE
res_rfe <- rfe(dengue2[,2:12], dengue2[,1], sizes=c(2:12), rfeControl=controlrfe)
#Resultados
print(res_rfe)
